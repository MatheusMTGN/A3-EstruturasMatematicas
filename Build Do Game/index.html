Bem-vindo ao Desafio

Este projeto é mais do que um jogo: é uma exploração interativa das Estruturas Matemáticas que governam o famoso Cubo Mágico. Criado na Unity, este aplicativo demonstra como conceitos abstratos de Álgebra Linear, Teoria de Conjuntos e Funções são aplicados para criar a lógica e a dinâmica de um objeto 3D complexo. Use o mouse para girar as faces e explore a beleza matemática em ação.

Os Pilares Matemáticos do Cubo

Matrizes e Vetores: A Arquitetura do Movimento

Cada peça do cubo tem uma posição no espaço 3D, representada por um vetor `(x, y, z)`. Quando você gira uma face, o programa aplica uma matriz de rotação a cada um dos vetores das peças naquela face. Essa operação de Álgebra Linear calcula a nova posição de cada peça de forma precisa, criando o movimento que você vê na tela.

Por exemplo, uma rotação de 90° em torno do eixo Z é representada pela matriz:

$$ R_z(90^\circ) = \begin{pmatrix} 0 & -1 & 0 \\ 1 & 0 & 0 \\ 0 & 0 & 1 \end{pmatrix} $$

Conjuntos e Funções: A Lógica do Quebra-Cabeça

O cubo é um conjunto de 26 peças. Cada movimento que você faz é uma função de permutação — uma regra que mapeia o estado atual do conjunto de peças para um novo estado. Resolver o cubo é encontrar a sequência correta de funções (movimentos) para reverter as permutações e retornar ao estado inicial ordenado.

Tutorial Rápido: Resolvendo por Camadas

Resolver o Cubo Mágico parece impossível, mas seguindo um método de camadas, torna-se um processo passo a passo. Aqui está a base do método para iniciantes:

Passo 1: A Cruz Branca

O primeiro objetivo é formar uma cruz na face branca, garantindo que as cores das peças laterais da cruz combinem com os centros das faces adjacentes. Isso é feito de forma intuitiva, sem algoritmos fixos.

Passo 2: Primeira Camada (Cantos Brancos)

Com a cruz pronta, posicione os quatro cantos da face branca. Encontre um canto com branco na sua face e posicione-o acima de onde ele deve ir. Use o algoritmo R U R' U' (Direita, Topo, Direita Invertido, Topo Invertido) repetidamente até que o canto se encaixe corretamente.

Passo 3: Segunda Camada (Peças do Meio)

Vire o cubo, deixando a face branca para baixo. Agora, encaixe as 4 peças das arestas do meio. Alinhe uma peça da camada de cima com seu centro correspondente e use um dos algoritmos abaixo para movê-la para a direita ou para a esquerda.

  • Para mover para a direita: U R U' R' U' F' U F
  • Para mover para a esquerda: U' L' U L U F U' F'

Passo 4: Última Camada (Face Amarela)

Esta é a etapa mais complexa, dividida em quatro partes:

  1. Fazer a Cruz Amarela: Use o algoritmo F R U R' U' F' até formar uma cruz amarela no topo.
  2. Orientar os Cantos Amarelos: Use R U R' U R U2 R' para fazer com que toda a face do topo fique amarela.
  3. Posicionar os Cantos: Use U R U' L' U R' U' L para colocar os cantos amarelos em suas posições finais (mesmo que a orientação esteja errada).
  4. Posicionar as Arestas: Use R U' R U R U R U' R' U' R2 para finalizar o cubo.

Pode parecer muito, mas com a prática, os algoritmos se tornam memória muscular. Boa sorte!
